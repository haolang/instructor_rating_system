### 文件结构说明

* css文件和js文件均放置在相应的文件夹下

* jslib放置插件，第三方库，框架等的js文件

* 文件根目录的PHP和HTML文件都是可以直接访问的页面

* php文件夹下的的php文件是为其他页面提供数据、拱其他页面请求、数据处理等，不能直接打开访问

* php/php_lib 文件下放置通用php文件，数据库连接，通用函数，插件，框架，第三方库等
